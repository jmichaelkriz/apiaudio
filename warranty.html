Warranty Registration

Warranty Information: Product carries a one year labor and five year parts warranty from original owner's date of purchase for any problem related to faulty parts. API does not cover claims for damage due to alteration, abuse, or abnormal wear and tear. This warranty is limited to failures during normal use, which are due to defects in material or workmanship.

Please note: The design or quality of any non-authorized third party service or vendor is beyond the control of API. The use of API modules in any NON-API rack or use of NON-API VPR Alliance modules in any API rack may VOID this warranty. Service or modification of any API unit except by an authorized API representative may VOID this warranty.

Repair Procedures: Should a repair be necessary, the unit must be shipped prepaid and in the original packaging; units not received in the original packaging are subject to a repackaging surcharge. Units will be returned by API prepaid, via UPS or Fed-Ex ground, unless upgrade options are paid for by the customer (overnight, two day option, etc.). Should a defect occur during the warranty period, contact API as soon as possible to have a Return Authorization number assigned. Returns will only be accepted with an RA number on the outside of the shipping box. Please submit a complete description of the problem when requesting an RA. Tape your name and RA number to the side of the unit before shipping.
